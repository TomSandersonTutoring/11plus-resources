Maths

The maths itself is the maths you already meet at school. What the 11+ really tests is how carefully you read the question, choose the right step, and set out your working.

Of the four subjects, maths is the most familiar, because it’s taught in class from the very start. That’s reassuring. The difference in the 11+ is that it asks for the same maths a little faster, and often tucks it inside a word problem, so the skill becomes as much about reading closely and picking the right method as about the sums themselves.

Number is by far the biggest part of the paper, so it’s worth feeling really sure there before anything else. Nothing below has to be done in order, though. Each topic explains itself from scratch, so start wherever you like.

Worth knowing: number does most of the work. In a typical paper there are far more number questions than any other kind, often around five times as many. So time spent getting quick and confident with the four operations, fractions, decimals, percentages and ratio is time very well spent.

How each topic works

1

Explanation

A plain, jargon-free description of the idea and the trick to it.

2

Worked example

One question taken slowly, step by step, so you can see the thinking.

3

Your go

A set to try yourself, with instant feedback that explains each answer.

Nothing here is timed, scored or saved. Get it wrong as many times as you like, that’s how it’s meant to work.
