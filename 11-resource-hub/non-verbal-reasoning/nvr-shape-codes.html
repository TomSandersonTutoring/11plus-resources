Shape codes

Every shape is given a short code of letters, where each letter stands for one feature. You are shown a few examples, work out what the letters mean, then use them. Everything you need is on this page.

Start here Horizontal codes Vertical codes

Start here

A shape code is a little label made of letters. Each letter describes one feature of the shape — perhaps what shape it is, how it is shaded, or how many there are. You are always given some example shapes with their codes. Those examples are the key: compare them and you can work out what each letter means, then read or build any code you like.

The trick is to take the letters one column at a time. Look only at the first letters across all the examples and ask what they have in common; then only the second letters. Each position is its own little puzzle.

  • Horizontal codes — the two letters sit side by side, and each stands for a feature.
  • Vertical codes — the same idea, but the two letters are stacked one above the other.

1. Horizontal codes

Here the code is two letters, side by side. In the examples below, the first letter tells you the shape and the second letter tells you the shading. Work out the key from the examples, then find the code for a new shape.

Worked example
Use the examples to work out the code for the new shape.
  1. First letters. A goes with the circle, B with the square, C with the triangle. So the first letter is the shape.
  2. Second letters. P goes with white, Q with shaded, R with solid. So the second letter is the shading.
  3. Build the new code. The new shape is a shaded circle: circle is A, shaded is Q.
Answer: AQ.

2. Vertical codes

Vertical codes work in just the same way, but the two letters are stacked, one above the other. To show you the idea is the same whatever the letters stand for, here the top letter tells you how many shapes there are, and the bottom letter tells you which shape.

Worked example
The top letter is the number, the bottom letter is the shape. Find the code.
  1. Top letters. J goes with one shape, K goes with two. So the top letter is how many.
  2. Bottom letters. A goes with the circle, B with the square, C with the triangle. So the bottom letter is the shape.
  3. Build the new code. The new picture is two circles: two is K, circle is A.
Answer: K above A.

Your go: vertical codes

Read the top letter as the number and the bottom letter as the shape, then tap the code for the new picture.

Which code is right?
