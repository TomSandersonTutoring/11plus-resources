Analogies & grids

The same idea in four costumes: work out the rule that links some pictures, then use it to find the one that is missing. Each type starts with a short explanation and a worked example.

Start here Analogies Matrices Similar triangles Star matrices

Start here

All four types on this page ask the same question in different ways: what is the rule, and what is missing? You are shown some pictures that follow a hidden rule, with one left blank. Find the rule, apply it, and pick the picture that fits the gap.

The rule is always one of the six things from the last topic — a change of shape, shading, size, number, position or rotation — so keep that check handy. The only new part is where to look: across a pair, along a row, down a column, or around a shape.

  • Analogies — “this changes to that, so this changes to…”
  • Matrices — a grid of pictures with one square empty.
  • Similar triangles — a pattern carried from one big triangle to another.
  • Star matrices — a pattern running around the points of a star.

1. Analogies

An analogy shows you one picture changing into another, then a third picture that must change in the very same way. It is written as A → B :: C → ? and read as “A is to B as C is to what?”. Work out what happened between A and B, then do exactly that to C.

Worked example
Work out how the first picture changes, then apply it to the third.
  1. Look at A → B. The arrow has turned a quarter-turn clockwise, from pointing up to pointing right. Nothing else changed.
  2. Name the rule. “Turn a quarter-turn clockwise.”
  3. Apply it to C. The triangle points up; a quarter-turn clockwise makes it point to the right.
Answer: the triangle pointing right.

2. Matrices

A matrix is a grid — usually two-by-two here — with one square left empty. The pictures change by a rule as you move across each row, and often the same is true down each column. Find the rule, then fill the empty square.

Worked example
The bottom-right square is missing. What belongs there?
  1. Read across the top row. The white circle becomes a solid circle. So the rule across is “the shape fills in solid.”
  2. Read down the columns. The circle becomes a square down the left column, so the shape changes from circle to square.
  3. Put it together. The bottom row is squares; the empty square is on the “solid” side. So it is a solid square.
Answer: a solid square.

3. Similar triangles

Here you are shown two large triangles, each split into four smaller ones. The second triangle follows the first by a rule, and one small piece is left blank. The commonest rule is the simplest: the second triangle copies the first, so the missing piece matches the same corner of the first triangle. Always check the pieces you can see first, to be sure it is a straight copy.

Worked example
The second triangle copies the first. Which piece is missing?
  1. Check the pieces you can see. Top, left and centre are the same in both triangles, so the second really is a copy.
  2. Find the blank. It is the bottom-right piece.
  3. Match it to the first triangle. In the first, the bottom-right piece is solid, so the missing piece is solid.
Answer: the solid triangle.

4. Star matrices

A star matrix has a picture at each of six points, with one point blank. The rule runs around the star. Two rules are common: the pictures may alternate as you go round, or points opposite each other may match. Check the pairs of opposite points first — it is the quickest way in.

Worked example
One point is blank. Opposite points match — what belongs there?
  1. Pair up opposite points. Top and bottom are both circles. Upper-right and lower-left are both squares.
  2. Find the blank’s partner. The blank is at the upper-left; its opposite point, the lower-right, is a triangle.
  3. Match it. Because opposite points are the same, the blank must be a triangle.
Answer: a triangle.
